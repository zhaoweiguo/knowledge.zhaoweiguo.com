2.5.2. 库加载¶

加载上级目录的库文件:

import sys
sys.path.append('..')
from common.utils import load_data

修改库文件后重新加载:

import importlib
from common import utils
importlib.reload(utils)

# 重新导入修改后的函数
from common.utils import mape
