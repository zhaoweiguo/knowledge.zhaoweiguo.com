е®ғе…·жңүеҶ…зҪ®зҡ„йҷҗйҖҹжңәеҲ¶пјҢжҺҘеҸЈд№ҹжҳ“дәҺдҪҝз”ЁгҖӮ
иҰҒдҪҝз”Ёе®ғпјҢеҝ…йЎ»еҫ—жҠҠeperзҡ„жүҖжңүдҫқиө–йғҪеўһеҠ иҝӣжқҘ

% ејәеӨ§зҡ„еҠЁжҖҒиҝҪиёӘеҮҪж•°пјҢеҸҜз”ЁдәҺеҠЁжҖҒжҢӮиҪҪй’©еӯҗгҖӮ
% 1. еҸҜжҢӮиҪҪжЁЎеқ—/еҮҪж•°и°ғз”Ё(з”ҡиҮіеҸҜеҜ№еҸӮж•°еҢ№й…Қ/иҝҮж»Ө)
% 2. еҸҜеҜ№и°ғз”ЁиҝӣзЁӢзӯӣйҖү(жҢҮе®ҡPidпјҢйҷҗеҲ¶ж–°е»әиҝӣзЁӢзӯү)
% 3. еҸҜйҷҗеҲ¶жү“еҚ°зҡ„иҝҪиёӘж•°йҮҸ/йҖҹзҺҮ
% 4. е…¶е®ғеҠҹиғҪпјҢеҰӮиҫ“еҮәйҮҚе®ҡеҗ‘пјҢиҝҪиёӘи°ғз”Ёз»“жһңзӯү

recon_trace:

жҳҜreconдёӯеӨ„зҗҶи·ҹиёӘзҡ„жЁЎеқ—
е®ғзҡ„зӣ®ж ҮжҳҜж—ўе…·жңүе’ҢredebugеҗҢж ·зә§еҲ«зҡ„дә§е“Ғе®үе…ЁжҖ§пјҢеҸҲдёҚйңҖиҰҒиҝҮеӨҡзҡ„дҫқиө–
е®ғеҸӘиғҪи·ҹиёӘеҮҪж•°и°ғз”ЁпјҢдёҚиғҪи·ҹиёӘж¶ҲжҒҜ

и·ҹиёӘеҺҹеҲҷ:

еҰӮжһңиҝӣзЁӢеҢ…еҗ«жңүж•Ҹж„ҹжҖ§жҒҜпјҢеҸҜд»Ҙи°ғз”Ёprocess_flag(sensitive,true)жқҘејәеҲ¶ж•°жҚ®з§Ғжңү

дҪҝз”ЁReconиҝӣиЎҢи·ҹиёӘ:

%% д»»дҪ•и°ғз”ЁqueueжЁЎеқ—зҡ„еҮҪж•°пјҢжңҖеӨҡжү“еҚ°10жқЎ:
recon_trace:calls({queue, вҖҷ_вҖҷ, вҖҷ_вҖҷ}, 10)

%% 100жқЎlists:seq(A,B):

%% дёҖз§ҚжөӢиҜ•ж— з”ЁиҪ¬еҢ–зҡ„ж–№жі•(еҺҹжқҘе°ұе·Із»ҸжҳҜbinaryзұ»еһӢдәҶ):

recon_trace:calls(TSpec, Max, [{args, arity}])

зјәзңҒжғ…еҶөдёӢпјҢformatter иҝӣзЁӢдјҡеңЁж”¶еҲ°зҡ„ж¶ҲжҒҜдёӯеўһеҠ ж—¶й—ҙжҲігҖӮеҰӮжһңйңҖиҰҒзІҫзЎ®зҡ„ж—¶й—ҙжҲіпјҢ еҸҜд»ҘйҖҡиҝҮйҖүйЎ№{timestamp, trace}ејәеҲ¶зӣҙжҺҘеңЁи·ҹиёӘж¶ҲжҒҜдёӯдҪҝз”Ёж—¶й—ҙжҲігҖӮ

жҳҜжү“еҚ°еҮәеҮҪж•°и°ғз”Ёзҡ„еҸӮж•°дёӘж•°пјҢиҝҳжҳҜжү“еҚ°еҮәе…¶еӯ—йқўиЎЁзӨә(зјәзңҒ)гҖӮ
