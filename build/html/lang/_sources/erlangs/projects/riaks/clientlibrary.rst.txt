################
Client Libraries
################

**Erlang**
    Riak HTTP and Protocol Buffers interfaces
    `riak-erlang-client on GitHub <https://github.com/basho/riak-erlang-client>`_
    `host the generated Erlang Client API Docs on GitHub. <http://basho.github.com/riak-erlang-client/>`_


**Javascript**
    用 `jquery <http://jquery.com/>`_ 通过ajax与riak进行交互。
    `riak-javascript-client on GitHub <https://github.com/basho/riak-javascript-client>`_
    `JavaScript sample app(yakriak——a Riak-powered Ajax-polling chatroom) <http://github.com/seancribbs/yakriak>`_

**Java**
    `riak-java-client repo on GitHub <http://github.com/basho/riak-java-client>`_
    `in the Sonatype Maven Repo <https://oss.sonatype.org/index.html#nexus-search;quick~riak>`_
    `benchmarks <http://wiki.basho.com/Java-Client-Benchmark.html>`_

**PHP**
    `riak-php-client repo on GitHub <https://github.com/basho/riak-php-client>`_
    `host the generated PHP client API Docs on GitHub <http://basho.github.com/riak-php-client/>`_
    `PHP sample app <http://github.com/schofield/riak_php_app>`_

**Python**
    `riak-python-client repo on GitHub <https://github.com/basho/riak-python-client>`_
    `host the generated Python client API Docs on GitHub <http://basho.github.com/riak-python-client/>`_
    `The Python Client is also available in the FreeBSD ports tree <http://www.freshports.org/databases/py-riak>`_

**Ruby**
    `officially supported Ruby driver for Riak(Ripple) <http://github.com/seancribbs/ripple>`_
    * `Documentation <http://seancribbs.github.com/ripple>`_
    * `Wiki <http://wiki.github.com/seancribbs/ripple>`_
    * `Client Gem <http://rubygems.org/gems/riak-client>`_
    * `Bug Tracker <http://github.com/seancribbs/ripple/issues>`_

    `Why Riak Should Power Your Next Rails App <http://seancribbs.com/tech/2010/02/06/why-riak-should-power-your-next-rails-app/>`_
