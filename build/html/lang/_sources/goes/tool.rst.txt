工具
########

mac 中安装golang调试器dlv::

    brew  install go-delve/delve/delve
    ln -s /usr/local/Cellar/delve/1.0.0/bin/dlv  /usr/local/bin/dlv



* `toml 文本转换问 golang 的 struct <https://xuri.me/toml-to-go/>`_
* `json 文本转换为 struct <https://mholt.github.io/json-to-go/>`_
* `sql 语句转换为 golang 的 struct <http://stming.cn/tool/sql2go.html>`_
* `curl 命令转化为具体的 golang 代码 <https://mholt.github.io/curl-to-go/>`_
