php数组
=============


* 用逗号分隔 键(key) => 值(value) 对::

    $arr = array("foo" => "bar", 12 => true);

::

   sort() - 以升序对数组排序
   rsort() - 以降序对数组排序
   asort() - 根据值，以升序对关联数组进行排序
   ksort() - 根据键，以升序对关联数组进行排序
   arsort() - 根据值，以降序对关联数组进行排序
   krsort() - 根据键，以降序对关联数组进行排序

   count($arr)      数组长度
   is_array($arr)   是否数组
