n命令
#####

安装::

	npm install -g n


使用::

	export N_PREFIX="$HOME/.n"		# 指定安装目录
	n lts    // 升级node.js到最新长期支持版



	n --help    // 使用帮助


	n ls 			// Output downloaded version
	n ls-remote		// Output matching versions available for download


	n uninstall                    Remove the installed Node.js
