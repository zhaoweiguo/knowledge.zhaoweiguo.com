vite
####

* Vite 是一个现代化的前端构建工具，由 Evan You（Vue.js 的作者）开发。Vite 的设计目标是 极快的开发启动速度 和 高效的生产构建。它通过 原生 ES 模块支持 和 按需构建 等技术，解决了传统构建工具在开发时的性能瓶颈，特别是在大规模项目中。
* Vite 借助 现代浏览器的特性，将构建流程和开发流程分开，大幅提高了开发体验。它支持多种前端框架，尤其优化了 Vue.js 和 React 的开发过程。

* 官网: https://cn.vite.dev/
* GitHub: https://github.com/vitejs/vite



安装
====

::

    npm install -g vite
