Parcel
######

* Parcel 是一个现代化的 零配置 JavaScript 打包工具，旨在提供开箱即用的开发体验。它支持 快速的模块打包、热模块替换（HMR） 和 代码分割 等功能，非常适合用来构建前端应用和网站。Parcel 不需要复杂的配置文件，就能自动识别项目类型并进行相应的优化，极大地简化了开发者的使用体验。
* Parcel 是一个现代化的、零配置的打包工具，特别适合快速构建前端项目。它通过提供自动化的依赖处理、支持多种资源类型、内置热模块替换等功能，使得开发者能够专注于编写代码而非配置文件。虽然它的插件系统不如 Webpack 强大，但对于中小型项目来说，Parcel 是一个非常高效且易于上手的选择。
