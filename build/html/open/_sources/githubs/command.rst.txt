github命令
##########

::

    /accept: 接受任务
    /delay-reset: 延长时间
    /pushed: 请求review



在 github 网站上快捷键::

    ?: 查看可用的快捷键
    t: 快速跳转到指定文件
    y: Update the URL to a permalink to the exact version of the file
    r: 定义跳转
    w: switch branch
    s: search
    l: jump to xxx line
    b: blame
