# ODC-By

* 许可全文: <https://opendatacommons.org/licenses/by/1-0/>
* ODC 官网: <https://opendatacommons.org/>



**ODC-By（Open Data Commons Attribution License）** 是一个开源数据许可协议，由 Open Knowledge Foundation 提出，属于 Open Data Commons（ODC）许可协议系列的一部分。其全名是：

> **Open Data Commons Attribution License (ODC-By)**

它类似于知识共享（Creative Commons）的 CC-BY 许可证，但专门用于数据库和数据集合的使用场景。

---

## 🔑 核心特点

ODC-By 的主要特点是：

| 特性                  | 描述                               |
| ------------------- | -------------------------------- |
| **署名（Attribution）** | 你可以自由使用、共享、修改、构建该数据库，但必须给出适当的署名。 |
| **开放性**             | 数据可以用于任何目的，包括商业用途。               |
| **保留自由**            | 可自由复制、发布、分发、传输和改编数据库。            |

---

## 📘 授权内容详解

根据 ODC-By 的规定，使用者获得以下权利：

* ✅ **复制权**：可复制数据库的全部或部分；
* ✅ **分发权**：可以传播数据库副本；
* ✅ **修改权**：可以创建衍生数据库（如重新整理、翻译、增删）；
* ✅ **商业使用**：可用于盈利目的。

但前提是：

> 必须明确注明数据库的原始来源、作者、版权信息和许可协议链接。

---

## 🧠 与其他协议比较

| 协议           | 使用自由度       | 是否要求署名 | 是否强制共享衍生作品 |
| ------------ | ----------- | ------ | ---------- |
| **ODC-PDDL** | 最大自由，类似 CC0 | ❌      | ❌          |
| **ODC-By**   | 自由但要求署名     | ✅      | ❌          |
| **ODC-ODbL** | 自由但强制共享     | ✅      | ✅          |

---

## ✅ 使用示例

当你使用 ODC-By 许可的数据时，推荐在你使用或发布成果中加入如下声明：

> 本项目使用了 \[原始数据集名称]，其由 \[作者] 提供，依据 [ODC-By 1.0 许可协议](https://opendatacommons.org/licenses/by/) 共享。
