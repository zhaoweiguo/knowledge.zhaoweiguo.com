# ODC-PDDL

* 协议说明页面：
  🌐 [https://opendatacommons.org/licenses/pddl/](https://opendatacommons.org/licenses/pddl/)

**ODC-PDDL（Open Data Commons Public Domain Dedication and License）** 是 Open Knowledge Foundation 提出的一个**完全开放的数据库许可协议**。它的目标是将数据库“贡献”到**公共领域（public domain）**，让任何人都可以**自由使用、修改、再发布，且不需要署名或遵守额外条件**。

---

## 🧾 ODC-PDDL 是什么？

全称：**Open Data Commons Public Domain Dedication and License**
缩写：**ODC-PDDL**

它模仿了 CC0（Creative Commons Zero）协议的精神，但专门用于数据库的数据集合，适用于法律上可以放弃版权的地区。

---

## 🔑 核心特点

| 特性           | 描述                          |
| ------------ | --------------------------- |
| ✅ **公共领域贡献** | 作者明确放弃数据库的所有版权与相关权利。        |
| ✅ **完全自由使用** | 任何人都可以复制、修改、分发、商用等，无需署名。    |
| ✅ **无附加义务**  | 不需要共享源数据、不需要注明来源。           |
| ✅ **适用于数据库** | 特别适合用于大规模开放数据的发布，如科研、政府数据等。 |

---

## 📜 权利详解

使用者拥有：

* **访问权**：可以自由访问数据库的全部内容；
* **复制与再分发权**：可以按任何方式分享数据库；
* **创建衍生作品的权利**：包括改编、重组、组合；
* **商业使用权**：可以用于盈利活动；
* **无署名义务**：甚至不需要提到原始作者。

> ⚠️ 唯一要求是：如果你确实使用了此协议授权的数据库，**不得虚假声称你拥有原数据库的版权**。

---

## ✅ 适用场景

ODC-PDDL 适用于发布者希望**最大程度促进数据自由传播和再利用**的情况，例如：

| 应用场景     | 示例             |
| -------- | -------------- |
| 政府开放数据平台 | 气象、地理、交通、统计等数据 |
| 公共科研项目   | 基因组数据库、观测数据集   |
| 开放数据倡议   | 公共文化项目、开放知识图谱  |

---

## 🆚 与其他开放数据协议对比

| 协议           | 是否署名 | 是否要求共享修改 | 适用场景                   |
| ------------ | ---- | -------- | ---------------------- |
| **ODC-PDDL** | ❌    | ❌        | 完全开放（公共领域）             |
| **ODC-By**   | ✅    | ❌        | 要署名但自由使用               |
| **ODbL**     | ✅    | ✅        | 强制共享衍生数据库              |
| **CC0**      | ❌    | ❌        | 内容/创作通用的公共领域协议，但不专注数据库 |

---

## ✍️ 使用声明建议（可选）

虽然使用者**不强制署名**，但发布者可以建议使用以下格式来尊重原始数据工作：

> 本数据库由 \[作者/机构名] 发布，依据 [ODC-PDDL 1.0](https://opendatacommons.org/licenses/pddl/) 协议授权为公共领域数据。你可以自由使用，无需署名。
