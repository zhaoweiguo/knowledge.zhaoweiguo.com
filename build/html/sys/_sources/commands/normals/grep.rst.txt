.. _grep:

grep使用方法
==================


::

    grep <字符串>|”<正则表达式>” [文件名]

    -o, --only-matching
      Prints only the matching part of the lines.


* 查找对应目录下的所有文件(包括子目录)中的内容::

    grep -R <keyword> <path>

* 使用正则需要加-E::

    grep -E <正则> <文件>


* 取出不含<string>的列表-v::

    <list> | grep -v <string>
    说明:
        可以来排除掉grep命令本身的输出
        ps aux | grep -v grep | grep <keyword>


* 不区分大小写::
  
    grep -i <keyword> <path>





实例::

    // 查询以"Aug 17"开头的内容
    grep "Aug 17.*" <file>

    $ curl www.zhaoweiguo.com/productpage | grep -o "<title>.*</title>"
    <title>Simple Bookstore App</title>

    // 查询文件中是否含有keyword, 有返回true, 否则返回false
    grep -q "keyword" <file>
