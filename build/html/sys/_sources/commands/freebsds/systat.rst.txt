systat命令
######################
::

    systat -- display system statistics

常用命令::

    systat -iostat    // 磁盘输入输出
    systat -ifstat  
      // Display the network traffic going through active interfaces on the system. 
      // 通过它可以看出带宽的占用率
      // 如10Mb带宽, 也就是说10/8MB/s的traffic
    systat -vmstat    // 
    systat -netstat   //
