fzf命令
=======

* A command-line fuzzy finder: https://github.com/junegunn/fzf

安装::

    $ brew install fzf

    源码安装
    $ git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
    $ ~/.fzf/install


用法::

    $ find * -type f | fzf > selected
