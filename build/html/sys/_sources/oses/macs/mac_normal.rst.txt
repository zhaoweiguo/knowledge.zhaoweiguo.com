路由转发功能
============

临时开启路由转发::

    sysctl -w net.inet.ip.forwarding=1

永久开启路由转发::

    $ sudo vim /etc/sysctl.conf:
    net.inet.ip.forwarding=1

    $ sysctl -p     // 使生效

查看 conf 变量::

    $ sudo sysctl -a | grep forward
    net.inet.ip.forwarding: 0
    net.inet6.ip6.forwarding: 0
