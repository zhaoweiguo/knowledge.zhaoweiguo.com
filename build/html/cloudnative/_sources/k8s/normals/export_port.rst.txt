暴露端口的方式
##############


暴露端口的方式有以下几种::

    NodePort（节点端口）
        将容器的端口映射到节点上的随机端口，可以通过节点的 IP 地址和该随机端口访问服务。

    ClusterIP（集群内部 IP）
        在集群内部创建一个虚拟 IP 地址，只能在集群内部访问服务。

    LoadBalancer（负载均衡器）
        通过云服务提供商或者硬件负载均衡器为服务分配一个外部 IP 地址，可以在公共网络上访问服务。

    Ingress（入口）
        使用 Ingress 资源配置规则，将外部流量路由到集群内的服务，
        支持基于 HTTP/HTTPS 的规则配置，是一种高级的流量管理方式。
