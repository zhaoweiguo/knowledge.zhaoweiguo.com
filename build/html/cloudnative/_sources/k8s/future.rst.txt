Cellery
============

* 介绍 [1]_
* github [2]_
* Cellery 语言基于allerina语言 [3]_
* Cellery 是围绕 DevOps 实践而设计的，因此，可以使用蓝 / 绿部署和金丝雀部署无缝地进行构建、推送、拉取、测试、部署和更新
* 简言之，Cellery 的目标是简化 Kubernetes 上应用程序的配置、构建、测试和部署。
