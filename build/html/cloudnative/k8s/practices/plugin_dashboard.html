3.5.4. dashboard服务安装¶

配置kubernetes-dashboard [1]¶

安装kubernetes-dashboard，并使得能够远程访问，yaml文件dashboard.yaml详见 [2] 执行:

$ kubectl apply -f dashboard.yaml
文件执行完成后，可以通过https://IP:30000 来访问（注意: 是https不是http）

在kubernetes 1.7之后建议使用token去登录
