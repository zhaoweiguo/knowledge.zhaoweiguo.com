sha256加密¶

实例:

# 验证网上下载包
// 1.下载安装文件
curl -LO https://fastdl.mongodb.org/osx/mongodb-osx-ssl-x86_64-3.4.9.tgz
// 2.下载sha256
curl -LO https://fastdl.mongodb.org/osx/mongodb-osx-ssl-x86_64-3.4.9.tgz.sha256
// 3.验证
shasum -c mongodb-osx-ssl-x86_64-3.4.9.tgz.sha256
