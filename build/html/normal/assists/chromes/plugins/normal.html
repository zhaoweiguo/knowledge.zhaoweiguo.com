常用¶

1.Octotree插件 Code tree for GitHub and GitLab GitLab code view插件: 专用于gitlab

2.clipboard history插件 保存你剪切版里面的内容

3.马甲插件 可快速切换同一网站的多个已登录的帐号，支持豆瓣、知乎、微博、twitter等

4.Context插件 可以对扩展进行管理

5.Color Tab 打开一个新标签页，会给你一个4色配色方案，方便设计师思枯意竭时找灵感

6.sourcegraph * Github版IDE

7.Tampermonkey插件 * Chrome上最流行的脚本管理工具 * 其他类似插件: 暴力猴 搜索

Github 镜像访问，加速下载
Toc Bar
