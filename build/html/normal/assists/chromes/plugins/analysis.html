数据分析¶

History Trends Unlimited¶

History Trends Unlimited 介绍 搜索和分析您的 Chrome 所有的历史记录，它的分析都是在本地，不会经过服务器。

推荐理由 了解你的行为习惯，了解你常访问的站点。

TimeYourWeb Time Tracker¶

介绍 最详细的在线时间统计。 让你能清楚的知道你每天在各个网站上时间开销。

推荐理由 废话不多说，我直接贴一个统计图吧。

Rooster for Chrome™¶

Rooster for Chrome™ 介绍 Rooster 帮助你保持高效，它能让你洞察自己的浏览习惯，并在你注意力不集中时提醒你。
