术语表
######

* strategy: n, 策略, 机制

* current: 当前
* paper: 论文
* arch: 架构

::

    mock: 模仿
    fake: 伪造

    arg 参数  argument

    avg 平均值 average
    bk  后退  back

    len 长度  length

    lnk 链接  link
    log 日志  log

    mem 内存  memory
    mgr 管理者 manager
    mid 中间  middle

    msg 信息  message
    mul 相乘  multiply
    num 数字  number

    sync  同步  synchronization
    sys 系统  system
    tbl 表 table
    temp  临时  temporary
    tst 测试  test
    txt 文本  text
    unk 未知  unknown
    upd 更新  update
    upg 升级  upgrade
    util  工具  util
    var 变量  variable
    ver 版本  version
    vert  垂直的 vertical

    wnd 窗口  window
