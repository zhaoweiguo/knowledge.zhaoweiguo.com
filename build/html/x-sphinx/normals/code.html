全局¶

功能:

1. 为行数超过5行的自动显示行数
即使不加:linenos:或:lineno-start:标签也显示行数
2. 默认一直使用console语法高亮直到遇到下一个highlight

不指定显示行数则不显示:

echo "1"
echo "1"

指定显示行数则显示:

1
2
echo "1"
echo "1"

超过5行的，不指定显示行数也会显示:

1
2
3
4
5
def some_function():
    interesting = False
    print 'This line is highlighted.'
    print 'This one is not...'
    print '...but this one is.'
