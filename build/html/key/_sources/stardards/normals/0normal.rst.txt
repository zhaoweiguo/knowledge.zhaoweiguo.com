常用
####


非编码类规范，主要包括::

    1. 开源规范
    2. 文档规范
    3. 版本规范
    4. Commit 规范
    5. 发布规范

编码类规范，则主要包括::

    1. 目录规范
    2. 代码规范
    3. 接口规范
    4. 日志规范
    5. 错误码规范

development lifecycle::
    
    1. Draft
    2. Experimental
    3. Stable
    4. Deprecated
    5. Removed

命名::

    驼峰命名法 (serverAddress)
    蛇形命名法 (server_address) 
    脊柱命名法 (server-address)
