2.3.5. Androidи§„иҢғВ¶

ж ҮиҜҶз¬Ұзҡ„е‘ҪеҗҚпјҢд»Ҙй©јеі°е‘ҪеҗҚжі•дёәдё»пјҢеҠӣжұӮеҒҡеҲ°з»ҹдёҖгҖҒиҫҫж„Ҹе’Ңз®ҖжҙҒгҖӮ

1.еҢ…еҗҚ:

1.1дҪҝз”Ёе°ҸеҶҷеӯ—жҜҚеҰӮ com.colorful.utilsпјҢдёҚиҰҒ com.Colorful.Utils
еҚ•иҜҚй—ҙдёҚиҰҒз”Ёеӯ—з¬Ұйҡ”ејҖпјҢжҜ”еҰӮ com.colorful.viewutilsпјҢиҖҢдёҚиҰҒcom.colorful.view_utils

1.2еҗҺзјҖ:
дҪҝз”ЁдёҚеҗҢзҡ„еҗҺзјҖиЎЁзӨәдёҚеҗҢзҡ„ж„ҸжҖқгҖӮжҜ”еҰӮпјҡ
жүҖжңүutilеҢ…йңҖж·»еҠ utilsеҗҺзјҖпјӣcom.colorful.utils

2.зұ»еҗҚ:

2.1жҜҸдёӘеҚ•иҜҚйҰ–еӯ—жҜҚеӨ§еҶҷпјҢжҜ”еҰӮColorfulUtils
2.2еҗҺзјҖ:дҪҝз”ЁдёҚеҗҢзҡ„еҗҺзјҖиЎЁзӨәдёҚеҗҢзҡ„ж„ҸжҖқпјҢжҜ”еҰӮ:

  жүҖжңүActivityзұ»еһӢзҡ„ж–Үд»¶еҗҚйңҖж·»еҠ ActivityеҗҺзјҖпјӣеҰӮпјҡXxxActivity.java
  жүҖжңүFragmentзұ»еһӢзҡ„ж–Үд»¶еҗҚйңҖж·»еҠ FragmentеҗҺзјҖпјӣеҰӮпјҡXxxFragment.java
  жүҖжңүAdapterзұ»еһӢзҡ„ж–Үд»¶еҗҚйңҖж·»еҠ AdapterеҗҺзјҖпјӣеҰӮпјҡXxxAdapter.java
  жүҖжңүutilеҢ…дёӯзҡ„ж–Үд»¶еҗҚйңҖж·»еҠ UtilsеҗҺзјҖпјӣеҰӮпјҡXxxUtils.java

3.ж–№жі•еҗҚ:

йҰ–еӯ—жҜҚе°ҸеҶҷпјҢеҰӮ addOrder() дёҚиҰҒ AddOrder()
еҠЁиҜҚеңЁеүҚпјҢеҰӮ addOrder()пјҢдёҚиҰҒorderAdd()

4.еҸҳйҮҸ:

4.1жүҖжңүзұ»дёӯзҡ„йқһеёёйҮҸжҲҗе‘ҳеҸҳйҮҸеқҮд»ҘmејҖеӨҙпјҢе№¶жҸҗдҫӣеӨҡиЎҢжіЁйҮҠ;
дҫӢеҰӮпјҡ

/**
 * ж ҮиҜҶжҳҜеҗҰдёәDog
 */
private boolean mIsDog;
4.2жүҖжңүжҺ§д»¶зұ»еһӢзҡ„еҸҳйҮҸпјҢеқҮйңҖж·»еҠ ж ҮиҜҶеёғеұҖзұ»еһӢзҡ„еүҚзјҖпјӣ

дҫӢеҰӮпјҡ

private TextView mTvContent; //(жҲҗе‘ҳеҸҳйҮҸ)
RelativeLayout rlContentBackground;// (еұҖйғЁеҸҳйҮҸ)
4.3еёёйҮҸжүҖжңүеӯ—жҜҚеӨ§еҶҷпјҢд»ҘдёӢеҲ’зәҝйҡ”ејҖпјҢе№¶жҸҗдҫӣеӨҡиЎҢжіЁйҮҠпјҢдҫӢеҰӮ

/**
* еӨҮжіЁжңҖеӨ§дёӘж•°
*/
private int REMARKS _MAX _SIZE=5;

5.еёғеұҖж–Үд»¶е‘ҪеҗҚи§„иҢғ:

Xmlж–Үд»¶дёҚиғҪеҢ…еҗ«еӨ§еҶҷеӯ—жҜҚпјҢеҚ•еӯҗд№Ӣй—ҙд»ҘдёӢеҲ’зәҝй—ҙйҡ”
5.1 Activityзұ»еёғеұҖж–Үд»¶пјҢд»Ҙactivity_еүҚзјҖејҖеӨҙпјӣ
5.2Fragmentзұ»еёғеұҖж–Үд»¶пјҢд»Ҙfragment_еүҚзјҖејҖеӨҙпјӣ
5.3Dialogе‘ҪеҗҚпјҡdialog_жҸҸиҝ°.xmlпјӣдҫӢеҰӮпјҡdialog_hint.xml
5.4PopupWindowе‘ҪеҗҚпјҡppw_жҸҸиҝ°.xmlпјӣдҫӢеҰӮпјҡppw_info.xml
5.5еҲ—иЎЁйЎ№е‘ҪеҗҚlistitem_жҸҸиҝ°.xmlпјӣдҫӢеҰӮпјҡlistitem_city.xml
5.6еҢ…еҗ«йЎ№пјҡinclude_жЁЎеқ—.xmlпјӣдҫӢеҰӮпјҡinclude_head.xmlгҖҒinclude_bottom.xml
5.7иҮӘе®ҡд№үеёғеұҖзұ»зҡ„еёғеұҖж–Үд»¶пјҢд»Ҙwidget_еүҚзјҖејҖеӨҙ

6.е…¶д»–и§„иҢғ:

6.1жүҖжңүзҡ„startActivityForResult()ж–№жі•пјҢж ҮжіЁиҜ·жұӮжқҘжәҗзҡ„requestCodeе’ҢresultCodeз»ҹдёҖд»ҘдёҚеҸҜеҸҳеёёйҮҸж·»еҠ еңЁеҗ„иҮӘеҜ№еә”зҡ„зұ»дёӯпјҢе№¶ж·»еҠ еӨҡиЎҢжіЁйҮҠ
6.2жүҖжңүе№ҝж’ӯжҺҘж”¶еҷЁең°еқҖж ҮиҜҶеӯ—з¬ҰдёІеёёйҮҸз»ҹдёҖж·»еҠ еңЁGlobalUtil.javaж–Үд»¶дёӯпјҢд»ҘBROADCAST_еүҚзјҖејҖе§ӢпјҢе№¶ж·»еҠ еӨҡиЎҢжіЁйҮҠ

дҫӢеҰӮ:

/**
  * жӣҙж–°еҫ…еҠһдәӢйЎ№зҡ„е№ҝж’ӯең°еқҖ
  */
public static final String BROADCAST_UPDATE_TODO = "com.szqd.calendar.todofragment.todolist_update_broadcast";
