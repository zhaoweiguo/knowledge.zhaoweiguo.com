2.1.7. зӣ®еҪ•з»“жһ„и§„иҢғВ¶

дёҖдёӘеҘҪзҡ„зӣ®еҪ•з»“жһ„иҮіе°‘иҰҒж»Ўи¶ід»ҘдёӢеҮ дёӘиҰҒжұӮ:

1. е‘ҪеҗҚжё…жҷ°
    зӣ®еҪ•е‘ҪеҗҚиҰҒжё…жҷ°гҖҒз®ҖжҙҒпјҢдёҚиҰҒеӨӘй•ҝпјҢд№ҹдёҚиҰҒеӨӘзҹӯпјҢзӣ®еҪ•еҗҚиҰҒиғҪжё…жҷ°ең°иЎЁиҫҫеҮәиҜҘзӣ®еҪ•е®һзҺ°зҡ„еҠҹиғҪпјҢе№¶дё”зӣ®еҪ•еҗҚжңҖеҘҪз”ЁеҚ•ж•°гҖӮ
    дёҖж–№йқўжҳҜеӣ дёәеҚ•ж•°и¶ід»ҘиҜҙжҳҺиҝҷдёӘзӣ®еҪ•зҡ„еҠҹиғҪпјҢеҸҰдёҖж–№йқўеҸҜд»Ҙз»ҹдёҖи§„иҢғпјҢйҒҝе…ҚеҚ•еӨҚж··з”Ёзҡ„жғ…еҶөгҖӮ
2. еҠҹиғҪжҳҺзЎ®
    дёҖдёӘзӣ®еҪ•жүҖиҰҒе®һзҺ°зҡ„еҠҹиғҪеә”иҜҘжҳҜжҳҺзЎ®зҡ„гҖҒе№¶дё”еңЁж•ҙдёӘйЎ№зӣ®зӣ®еҪ•дёӯе…·жңүеҫҲй«ҳзҡ„иҫЁиҜҶеәҰ
    д№ҹе°ұжҳҜиҜҙпјҢеҪ“йңҖиҰҒж–°еўһдёҖдёӘеҠҹиғҪж—¶пјҢжҲ‘д»¬иғҪеӨҹйқһеёёжё…жҘҡең°зҹҘйҒ“жҠҠиҝҷдёӘеҠҹиғҪж”ҫеңЁе“ӘдёӘзӣ®еҪ•дёӢ
3. е…ЁйқўжҖ§
    зӣ®еҪ•з»“жһ„еә”иҜҘе°ҪеҸҜиғҪе…Ёйқўең°еҢ…еҗ«з ”еҸ‘иҝҮзЁӢдёӯйңҖиҰҒзҡ„еҠҹиғҪпјҢ
    дҫӢеҰӮж–ҮжЎЈгҖҒи„ҡжң¬гҖҒжәҗз Ғз®ЎзҗҶгҖҒAPI е®һзҺ°гҖҒе·Ҙе…·гҖҒз¬¬дёүж–№еҢ…гҖҒжөӢиҜ•гҖҒзј–иҜ‘дә§зү©зӯүгҖӮ
4. еҸҜйў„жөӢжҖ§
    йЎ№зӣ®и§„жЁЎдёҖе®ҡжҳҜд»Һе°ҸеҲ°еӨ§зҡ„пјҢжүҖд»ҘдёҖдёӘеҘҪзҡ„зӣ®еҪ•з»“жһ„еә”иҜҘиғҪеӨҹеңЁйЎ№зӣ®еҸҳеӨ§ж—¶пјҢд»Қз„¶дҝқжҢҒд№ӢеүҚзҡ„зӣ®еҪ•з»“жһ„
5. еҸҜжү©еұ•жҖ§
    жҜҸдёӘзӣ®еҪ•дёӢеӯҳж”ҫдәҶеҗҢзұ»зҡ„еҠҹиғҪпјҢеңЁйЎ№зӣ®еҸҳеӨ§ж—¶пјҢиҝҷдәӣзӣ®еҪ•еә”иҜҘеҸҜд»Ҙеӯҳж”ҫжӣҙеӨҡеҗҢзұ»еҠҹиғҪ

зӣ®еҪ•з»“жһ„еҲҶдёәз»“жһ„еҢ–зӣ®еҪ•з»“жһ„е’Ңе№ій“әејҸзӣ®еҪ•з»“жһ„дёӨз§Қ:

з»“жһ„еҢ–зӣ®еҪ•з»“жһ„дё»иҰҒз”ЁеңЁ Go еә”з”ЁдёӯпјҢзӣёеҜ№жқҘиҜҙжҜ”иҫғеӨҚжқӮпјӣ
иҖҢе№ій“әејҸзӣ®еҪ•з»“жһ„дё»иҰҒз”ЁеңЁ Go еҢ…дёӯпјҢзӣёеҜ№жқҘиҜҙжҜ”иҫғз®ҖеҚ•гҖӮ
