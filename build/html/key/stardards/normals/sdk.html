2.1.16. SDK и§„иҢғВ¶

еёёи§Ғзҡ„е‘ҪеҗҚж–№ејҸ:

xxx-sdk-go / xxx-sdk-python / xxx-sdk-java

дёҖдёӘе…ёеһӢзҡ„зӣ®еҪ•з»“жһ„еҰӮдёӢ:

в”ңв”Җв”Җ examples            # зӨәдҫӢд»Јз Ғеӯҳж”ҫзӣ®еҪ•
в”Ӯ   в””в”Җв”Җ authz.go
в”ңв”Җв”Җ README.md           # SDKдҪҝз”Ёж–ҮжЎЈ
в”ңв”Җв”Җ sdk                 # е…¬е…ұеҢ…пјҢе°ҒиЈ…дәҶSDKй…ҚзҪ®гҖҒAPIиҜ·жұӮгҖҒи®ӨиҜҒзӯүд»Јз Ғ
в”Ӯ   в”ңв”Җв”Җ client.go
в”Ӯ   в”ңв”Җв”Җ config.go
в”Ӯ   в”ңв”Җв”Җ credential.go
в”Ӯ   в””в”Җв”Җ ...
в””в”Җв”Җ services            # APIе°ҒиЈ…
    в”ңв”Җв”Җ common
    в”Ӯ   в””в”Җв”Җ model
    в”ңв”Җв”Җ iam             # iamжңҚеҠЎзҡ„APIжҺҘеҸЈ
    в”Ӯ   в”ңв”Җв”Җ authz.go
    в”Ӯ   в”ңв”Җв”Җ client.go
    в”Ӯ   в””в”Җв”Җ ...
    в””в”Җв”Җ tms             # tmsжңҚеҠЎзҡ„APIжҺҘеҸЈ
