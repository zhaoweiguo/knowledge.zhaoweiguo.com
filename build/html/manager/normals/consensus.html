1.3. 团队的共识¶

团队的共识分很多层次:

1. 其一，团队是不是有共同的目标
2. 其二，团队是不是有共同的行事做人的准则
3. 其三，对产品与市场的要与不要，以及为什么要或为什么不要，是否已达成一致
4. 其四，对执行路径有没有共同的认知
5. 其五，有没有团队默契，是否日常沟通交流很多地方不必赘述，沟通上一点即透

共识是为了更高的决策效率，所以共识可以是使命愿景价值观，可以是决策原则，等等。
