Memento-备忘录¶

  • 备忘录模式，也叫快照（Snapshot）模式

Note

provides the ability to restore an object to its previous state (undo). Captures and externalizes an object’s internal state so that it can be restored later, all without violating encapsulation. 【定义】在不违背封装原则的前提下，捕获一个对象的内部状态，并在该对象之外保存这个状态，以便之后恢复对象为先前的状态。

应用场景也比较明确和有限，主要是用来防丢失、撤销、恢复等。

这个模式的定义主要表达了两部分内容:

1. 存储副本以便后期恢复。这一部分很好理解
2. 要在不违背封装原则的前提下，进行对象的备份和恢复

与”备份”的区别¶

备忘录模式更侧重于代码的设计和实现，备份更侧重架构设计或产品设计
