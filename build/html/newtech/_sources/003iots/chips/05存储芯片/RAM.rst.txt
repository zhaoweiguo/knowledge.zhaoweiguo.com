RAM
###

RAM：随机存取存储器，也叫主存，是与 CPU 直接交换数据的内部存储器，可以随时读写，而且速度快，通常作为操作系统或其他正在运行程序的临时数据存储介质。RAM 工作时可以随时从任何一个指定地址写入或读出信息。他与 ROM 的最大区别是数据的易失性，断电丢失。RAM 在计算机和数字系统中用来暂时存储程序、数据和中间结果。手机和电脑的运行内存都是使用 ram 为存储空间，内存条的作用是增加运行 ram 空间。

SRAM：静态随机存储器，SRAM 存放的信息在不停电的情况下能长时间保留，状态稳定，不需外加刷新电路，从而简化了外部电路设计。常作为 Cache。

DRAM：动态随机存储器，DRAM 与 SRAM 相比具有集成度高、功耗低、价格便宜等优点，所以在大容量存储器中普遍采用。DRAM 的缺点是需要刷新逻辑电路，且刷新操作时不能进行正常读，写操作。常作为主存储器。

SDRAM：同步动态随机存取内存。就是 DDR，被分为 DDR1、DDR2、DDR3。
