####################

::

  // 测试版
  $ geth --testnet console
  // proof-of-authority based test network
  $ geth --rinkeby console


* The clique system is also used for several public Ethereum testnets such as ``Rinkeby`` and ``Görli``.

* Ethash consensus, being a proof-of-work algorithm, is a system that allows open participation by anyone willing to dedicate resources to mining. 
* Clique consensus is a proof-of-authority system where new blocks can be created by authorized ‘signers’ only. The clique consenus protocol is specified in `EIP-225 <https://eips.ethereum.org/>`_
