жҺЁжөҒ&жӢүжөҒВ¶

1. йҖҡиҝҮж‘„еғҸеӨҙжҲ–еҪ•еұҸеҠҹиғҪиҝӣиЎҢи§Ҷйў‘йҮҮйӣҶи§Ҷйў‘еҺҹе§Ӣж•°жҚ®ж јејҸ: ``RGB`` жҲ– ``YUV``
2. йҖҡиҝҮиҜқзӯ’иҝӣиЎҢйҹійў‘йҮҮйӣҶйҹійў‘еҺҹе§Ӣж•°жҚ®ж јејҸ: ``PCM``
3. е°Ҷи§Ҷйў‘ж–Үд»¶зј–з Ғи§Ҷйў‘ж јејҸд»ҺYUV(RGB)еҸҳдёә ``H.264(H.265)``
4. е°Ҷйҹійў‘ж–Үд»¶зј–з Ғйҹійў‘ж јејҸд»Һ PCM еҸҳдёә ``AAC``
5. е°Ҷи§Ҷйў‘дёҺйҹійў‘ж–Үд»¶зј–з ҒдёәжңүвҖңжөҒеӘ’дҪ“вҖқзү№жҖ§зҡ„еӨҡеӘ’дҪ“е®№еҷЁж јејҸ(Multimedia Container Format)
    жӯӨж—¶йҹійў‘е’Ңи§Ҷйў‘ (H.264е’ҢAAC) з»“еҗҲжҲҗдәҶ ( ``FLV`` жҲ– ``TS`` жҲ– ``RTMP Packet``пјҢиҝҷиҰҒи§Ҷдј иҫ“еҚҸи®®зҡ„и§„е®ҡиҖҢе®ҡ)
6. йҖүжӢ©дёҖз§Қеҗ‘жңҚеҠЎеҷЁжҺЁжөҒзҡ„еҚҸи®®(вҖңжөҒвҖқеҚівҖңжңүжөҒеӘ’дҪ“зү№жҖ§зҡ„еӨҡеӘ’дҪ“е®№еҷЁж јејҸзҡ„ж–Үд»¶вҖқ)
    еә”з”ЁеұӮеҚҸи®®: HLS; RTSP; RTMP
    дј иҫ“еұӮеҚҸи®®: RTCPгҖҒRTP
    зҪ‘з»ңеұӮеҚҸи®®: RSVP

NVRВ¶

1. IPж‘„еғҸеӨҙ
    иҝҷдәӣжҳҜиҝһжҺҘеҲ°зҪ‘з»ңзҡ„ж•°еӯ—ж‘„еғҸжңәпјҢеҸҜд»ҘйҖҡиҝҮзҪ‘з»ңеҸ‘йҖҒи§Ҷйў‘е’Ңйҹійў‘ж•°жҚ®гҖӮ
    е®ғд»¬йҖҡеёёдҪҝз”ЁеҹәдәҺIPзҡ„зҪ‘з»ңеҚҸи®®пјҢеҰӮTCP/IPжҲ–UDP/IPиҝӣиЎҢйҖҡдҝЎгҖӮ
2. зҪ‘з»ңдәӨжҚўжңә
    иҝҷдәӣи®ҫеӨҮз”ЁдәҺиҝһжҺҘжүҖжңүзҡ„IPж‘„еғҸеӨҙе’ҢNVRжңҚеҠЎеҷЁгҖӮ
    зҪ‘з»ңдәӨжҚўжңәиҙҹиҙЈз®ЎзҗҶе’ҢиҪ¬еҸ‘зҪ‘з»ңжөҒйҮҸпјҢд»ҘзЎ®дҝқж•°жҚ®еҸҜд»ҘеңЁеҗ„дёӘи®ҫеӨҮд№Ӣй—ҙеҝ«йҖҹгҖҒе®үе…Ёең°дј иҫ“гҖӮ
3. NVRжңҚеҠЎеҷЁ
    иҝҷжҳҜдёҖдёӘдё“з”Ёзҡ„и®Ўз®—жңәжҲ–жңҚеҠЎеҷЁпјҢз”ЁдәҺеҪ•еҲ¶гҖҒеӯҳеӮЁе’Ңз®ЎзҗҶжүҖжңүд»ҺIPж‘„еғҸеӨҙжҺҘж”¶зҡ„и§Ҷйў‘е’Ңйҹійў‘ж•°жҚ®гҖӮ
    NVRжңҚеҠЎеҷЁйҖҡеёёе…·жңүй«ҳжҖ§иғҪзҡ„зЎ¬д»¶з»„д»¶е’Ңдё“з”Ёзҡ„иҪҜд»¶еә”з”ЁзЁӢеәҸпјҢеҸҜд»ҘжҸҗдҫӣејәеӨ§зҡ„и§Ҷйў‘зј–и§Јз ҒгҖҒзҪ‘з»ңжөҒйҮҸз®ЎзҗҶгҖҒеӯҳеӮЁз®ЎзҗҶе’Ңж•°жҚ®еӨҮд»ҪзӯүеҠҹиғҪгҖӮ
4. еӯҳеӮЁи®ҫеӨҮ
    иҝҷдәӣи®ҫеӨҮз”ЁдәҺеӯҳеӮЁжүҖжңүеҪ•еҲ¶зҡ„и§Ҷйў‘е’Ңйҹійў‘ж•°жҚ®гҖӮ
    еӯҳеӮЁи®ҫеӨҮеҸҜд»ҘжҳҜзЎ¬зӣҳй©ұеҠЁеҷЁгҖҒзҪ‘з»ңеӯҳеӮЁи®ҫеӨҮпјҲNASпјүжҲ–дә‘еӯҳеӮЁжңҚеҠЎзӯүгҖӮ
5. е®ўжҲ·з«ҜиҪҜд»¶
    иҝҷдәӣжҳҜеңЁз”ЁжҲ·и®Ўз®—жңәжҲ–з§»еҠЁи®ҫеӨҮдёҠе®үиЈ…зҡ„еә”з”ЁзЁӢеәҸпјҢз”ЁдәҺи®ҝй—®е’Ңз®ЎзҗҶNVRжңҚеҠЎеҷЁдёҠеӯҳеӮЁзҡ„и§Ҷйў‘е’Ңйҹійў‘ж•°жҚ®гҖӮ
    е®ўжҲ·з«ҜиҪҜд»¶йҖҡеёёе…·жңүеӣҫеҪўз”ЁжҲ·з•ҢйқўпјҲGUIпјүпјҢеҸҜд»Ҙи®©з”ЁжҲ·ж–№дҫҝең°жҗңзҙўгҖҒж’ӯж”ҫгҖҒдёӢиҪҪе’ҢеӨҮд»ҪеҪ•еҲ¶зҡ„ж•°жҚ®гҖӮ

IPCВ¶

  • зҪ‘з»ңж‘„еғҸжңәдёҖиҲ¬з”ұй•ңеӨҙгҖҒеӣҫеғҸдј ж„ҹеҷЁгҖҒеЈ°йҹідј ж„ҹеҷЁгҖҒдҝЎеҸ·еӨ„зҗҶеҷЁгҖҒ A/D иҪ¬жҚўеҷЁгҖҒзј–з ҒиҠҜзүҮгҖҒдё»жҺ§иҠҜзүҮгҖҒзҪ‘з»ңеҸҠжҺ§еҲ¶жҺҘеҸЈзӯүйғЁеҲҶз»„жҲҗгҖӮ

  • зҪ‘з»ңж‘„еғҸжңәеҸҲеҸ« IP CAMERAпјҲз®Җз§° IPCпјүз”ұ``зҪ‘з»ңзј–з ҒжЁЎеқ—``е’Ң``жЁЎжӢҹж‘„еғҸжңә``з»„еҗҲиҖҢжҲҗгҖӮзҪ‘з»ңзј–з ҒжЁЎеқ—е°ҶжЁЎжӢҹж‘„еғҸжңәйҮҮйӣҶеҲ°зҡ„жЁЎжӢҹи§Ҷйў‘дҝЎеҸ·зј–з ҒеҺӢзј©жҲҗж•°еӯ—дҝЎеҸ·пјҢд»ҺиҖҢеҸҜд»ҘзӣҙжҺҘжҺҘе…ҘзҪ‘з»ңдәӨжҚўеҸҠи·Ҝз”ұи®ҫеӨҮгҖӮзҪ‘з»ңж‘„еғҸжңәеҶ…зҪ®дёҖдёӘеөҢе…ҘејҸиҠҜзүҮпјҢйҮҮз”ЁеөҢе…ҘејҸе®һж—¶ж“ҚдҪңзі»з»ҹгҖӮ

  • зҪ‘з»ңж‘„еғҸжңәжҳҜдј з»ҹж‘„еғҸжңәдёҺзҪ‘з»ңи§Ҷйў‘жҠҖжңҜзӣёз»“еҗҲзҡ„ж–°дёҖд»Јдә§е“ҒгҖӮж‘„еғҸжңәдј йҖҒжқҘзҡ„и§Ҷйў‘дҝЎеҸ·ж•°еӯ—еҢ–еҗҺз”ұй«ҳж•ҲеҺӢзј©иҠҜзүҮеҺӢзј©пјҢйҖҡиҝҮзҪ‘з»ңжҖ»зәҝдј йҖҒеҲ° Web жңҚеҠЎеҷЁгҖӮзҪ‘з»ңдёҠз”ЁжҲ·еҸҜд»ҘзӣҙжҺҘз”ЁжөҸи§ҲеҷЁи§ӮзңӢ Web жңҚеҠЎеҷЁдёҠзҡ„ж‘„еғҸжңәеӣҫеғҸпјҢжҺҲжқғз”ЁжҲ·иҝҳеҸҜд»ҘжҺ§еҲ¶ж‘„еғҸжңәдә‘еҸ°й•ңеӨҙзҡ„еҠЁдҪңжҲ–еҜ№зі»з»ҹй…ҚзҪ®иҝӣиЎҢж“ҚдҪңгҖӮ

зҪ‘з»ңж‘„еғҸжңәзҡ„еҹәжң¬еҺҹзҗҶ:

1. еӣҫеғҸдҝЎеҸ·з»ҸиҝҮй•ңеӨҙиҫ“е…ҘеҸҠеЈ°йҹідҝЎеҸ·з»ҸиҝҮйәҰе…ӢйЈҺиҫ“е…ҘеҗҺпјҢз”ұеӣҫеғҸдј ж„ҹеҷЁе’ҢеЈ°йҹідј ж„ҹеҷЁиҪ¬еҢ–дёәз”өдҝЎеҸ·
2. A/D иҪ¬жҚўеҷЁе°ҶжЁЎжӢҹз”өдҝЎеҸ·иҪ¬жҚўдёәж•°еӯ—з”өдҝЎеҸ·
3. зј–з ҒеҷЁжҢүдёҖе®ҡзҡ„зј–з Ғж ҮеҮҶиҝӣиЎҢзј–з ҒеҺӢзј©
4. жҺ§еҲ¶еҷЁзҡ„жҺ§еҲ¶дёӢпјҢз”ұзҪ‘з»ңжңҚеҠЎеҷЁжҢүдёҖе®ҡзҡ„зҪ‘з»ңеҚҸи®®йҖҒдёҠеұҖеҹҹзҪ‘жҲ– INTERNET

зҪ‘з»ңж‘„еғҸжңәзҡ„еӣҫеғҸеҺӢзј©зј–з Ғж ҮеҮҶ:

MPEG4гҖҒ H.264гҖҒM-JPEG

ж‘„еғҸеӨҙж”ҜжҢҒи§Ҷйў‘жөҒзҡ„дј иҫ“пјҢйҖҡеёёжңүд»ҘдёӢдёӨз§Қж–№ејҸ:

RTSPеҚҸи®®дј иҫ“пјҡRTSPпјҲReal Time Streaming ProtocolпјүжҳҜдёҖз§ҚеҹәдәҺTCPжҲ–UDPзҡ„е®һж—¶жөҒеӘ’дҪ“еҚҸи®®пјҢжҳҜи§Ҷйў‘жөҒдј иҫ“зҡ„дё»жөҒеҚҸи®®д№ӢдёҖгҖӮ
ж‘„еғҸеӨҙеҸҜд»ҘйҖҡиҝҮRTSPеҚҸи®®е°Ҷи§Ҷйў‘жөҒдј иҫ“еҲ°NVRиҪҜд»¶жҲ–е…¶д»–и§Ҷйў‘жөҒжҺҘж”¶з«ҜгҖӮ
йҖҡеёёжғ…еҶөдёӢпјҢж‘„еғҸеӨҙдјҡжҸҗдҫӣдёҖдёӘRTSP URLпјҢз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮиҜҘURLзӣҙжҺҘи®ҝй—®ж‘„еғҸеӨҙзҡ„и§Ҷйў‘жөҒгҖӮ

ONVIFеҚҸи®®дј иҫ“пјҡONVIFпјҲOpen Network Video Interface ForumпјүжҳҜдёҖдёӘејҖж”ҫзҡ„зҪ‘з»ңи§Ҷйў‘жҺҘеҸЈж ҮеҮҶпјҢж—ЁеңЁдёәеҗ„з§ҚзҪ‘з»ңи§Ҷйў‘и®ҫеӨҮжҸҗдҫӣдә’ж“ҚдҪңжҖ§гҖӮ
йҖҡиҝҮONVIFеҚҸи®®пјҢж‘„еғҸеӨҙеҸҜд»Ҙе°Ҷи§Ҷйў‘жөҒдј иҫ“еҲ°ONVIFе…је®№зҡ„и®ҫеӨҮпјҢдҫӢеҰӮNVRиҪҜд»¶гҖҒи§Ҷйў‘зӣ‘жҺ§иҪҜд»¶зӯүгҖӮ
ONVIFеҚҸи®®ж”ҜжҢҒеӨҡз§Қи§Ҷйў‘зј–з Ғж јејҸпјҢдҫӢеҰӮH.264гҖҒMPEG-4зӯүгҖӮ

ејҖеҸ‘дёҖдёӘNVRиҪҜд»¶йҖҡеёёйңҖиҰҒд»ҘдёӢжӯҘйӘӨ:

зЎ®е®ҡйңҖжұӮпјҡйҰ–е…ҲйңҖиҰҒжҳҺзЎ®NVRиҪҜд»¶зҡ„еҠҹиғҪе’ҢйңҖжұӮпјҢдҫӢеҰӮж”ҜжҢҒзҡ„ж‘„еғҸеӨҙзұ»еһӢе’Ңж•°йҮҸгҖҒи§Ҷйў‘еҪ•еҲ¶е’ҢеӯҳеӮЁж–№ејҸгҖҒиҝңзЁӢзӣ‘жҺ§е’Ңз®ЎзҗҶеҠҹиғҪзӯүгҖӮ
и®ҫи®Ўжһ¶жһ„пјҡж №жҚ®йңҖжұӮе’ҢеҠҹиғҪпјҢи®ҫи®ЎNVRиҪҜд»¶зҡ„жһ¶жһ„е’ҢжЁЎеқ—пјҢдҫӢеҰӮи§Ҷйў‘жөҒжҺҘж”¶е’ҢеӨ„зҗҶжЁЎеқ—гҖҒеҪ•еҲ¶е’ҢеӯҳеӮЁжЁЎеқ—гҖҒиҝңзЁӢзӣ‘жҺ§е’Ңз®ЎзҗҶжЁЎеқ—зӯүгҖӮ
е®һзҺ°ж ёеҝғеҠҹиғҪпјҡж №жҚ®жһ¶жһ„е’ҢжЁЎеқ—и®ҫи®ЎпјҢе®һзҺ°NVRиҪҜд»¶зҡ„ж ёеҝғеҠҹиғҪпјҢдҫӢеҰӮи§Ҷйў‘жөҒжҺҘж”¶е’ҢеӨ„зҗҶгҖҒи§Ҷйў‘еҪ•еҲ¶е’ҢеӯҳеӮЁгҖҒиҝңзЁӢзӣ‘жҺ§е’Ңз®ЎзҗҶзӯүгҖӮ
UIи®ҫи®Ўе’Ңе®һзҺ°пјҡж №жҚ®йңҖжұӮе’Ңз”ЁжҲ·дҪ“йӘҢпјҢи®ҫи®Ўе’Ңе®һзҺ°NVRиҪҜд»¶зҡ„з”ЁжҲ·з•ҢйқўпјҢд»Ҙдҫҝз”ЁжҲ·еҸҜд»Ҙж–№дҫҝең°дҪҝз”Ёе’Ңз®ЎзҗҶNVRиҪҜд»¶гҖӮ
жөӢиҜ•е’ҢдјҳеҢ–пјҡиҝӣиЎҢеҠҹиғҪжөӢиҜ•е’ҢжҖ§иғҪжөӢиҜ•пјҢдјҳеҢ–NVRиҪҜд»¶зҡ„жҖ§иғҪе’ҢзЁіе®ҡжҖ§пјҢзЎ®дҝқиҪҜд»¶иғҪеӨҹзЁіе®ҡиҝҗиЎҢе’Ңж»Ўи¶із”ЁжҲ·йңҖжұӮгҖӮ
еҸ‘еёғе’Ңз»ҙжҠӨпјҡеҸ‘еёғNVRиҪҜд»¶пјҢе№¶иҝӣиЎҢеҗҺз»ӯзҡ„з»ҙжҠӨе’Ңжӣҙж–°пјҢд»ҘдҝқиҜҒиҪҜд»¶зҡ„зЁіе®ҡжҖ§е’ҢеҠҹиғҪе®Ңе–„жҖ§гҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢејҖеҸ‘NVRиҪҜд»¶йңҖиҰҒж¶үеҸҠеҲ°и§Ҷйў‘жөҒдј иҫ“гҖҒзј–и§Јз ҒгҖҒеӯҳеӮЁе’ҢзҪ‘з»ңйҖҡдҝЎзӯүж–№йқўзҡ„зҹҘиҜҶпјҢйңҖиҰҒжңүдёҖе®ҡзҡ„жҠҖжңҜз§ҜзҙҜе’Ңз»ҸйӘҢж”ҜжҢҒгҖӮ
еҗҢж—¶пјҢNVRиҪҜд»¶зҡ„ејҖеҸ‘д№ҹйңҖиҰҒиҖғиҷ‘еҲ°е®үе…Ёе’Ңйҡҗз§ҒдҝқжҠӨзӯүй—®йўҳпјҢзЎ®дҝқи§Ҷйў‘ж•°жҚ®дёҚдјҡиў«йқһжі•и®ҝй—®е’Ңжі„йңІгҖӮ
