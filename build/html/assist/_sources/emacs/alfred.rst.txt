alfred工具
================

最常用功能: File Search::


  1.search
  空格('): 快速启用打开文件或者文件夹功能类似于使用open关键字
  open: 打开文件或者文件夹
  find: 查询文件或者文件夹的位置
  in: 查找文本文件内含有查询文字的文件
  tags: 有查询tags的文件或者文件夹

  2.Navigation
  Option + Command + /
  关键词: previous

  3.Buffer选项
  Option + Command + \
  option键 + ⬆️来将选中的文件夹或者文件加入到缓存
  option键 + ➡️来批量处理缓存中的文件夹和文件
  
  4.Advanced选项
  Shift: 快速查看
  
最常用功能: Clipboard剪切板::

  1.History选项:
  Option + Command + c
  Viewer关键词: clipboard
  Snippet关键词: snip
  Clear关键词: clear



最常用功能:System::

  screensaver: 屏保
  trash: 垃圾箱
  emptytrash: 清空垃圾箱
  lock: 锁屏
  restart: 重启
  shutdown:关机
  sleep: 休眠

最常用功能:Snippets::

  Viewer Hotkey: Option + Command + v
  Snippet Keyword:snip
