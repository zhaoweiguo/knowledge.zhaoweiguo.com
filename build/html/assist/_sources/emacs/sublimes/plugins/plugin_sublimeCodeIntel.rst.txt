SublimeCodeIntel插件
####################

作用::

    代码间调转

Install CodeIntelPermalink::

    $ pip install -U CodeIntel

Install SublimeCodeIntelPermalink::

    打开command palette
    键入: Package Control: Install Package.
    输入 SublimeCodeIntel

相关目录::

    ~/.codeintel/

Shortcuts for jump to definition have changed::
  
    For Mac OS X:
      * Jump to definition = ``Control+Click``
      * Jump to definition = ``Control+Command+Alt+Up``
      * Go back = ``Control+Command+Alt+Left``
      * Manual CodeIntel = ``Control+Shift+space``
  
    For Linux:
      * Jump to definition = ``Super+Click``
      * Jump to definition = ``Control+Super+Alt+Up``
      * Go back = ``Control+Super+Alt+Left``
      * Manual CodeIntel = ``Control+Shift+space``
  
    For Windows:
      * Jump to definition = ``Alt+Click``
      * Jump to definition = ``Control+Windows+Alt+Up``
      * Go back = ``Control+Windows+Alt+Left``
      * Manual CodeIntel = ``Control+Shift+space``
  
  
* Online: https://www.sublimecodeintel.com/docs/welcome/
