nginx简介
#########

什么是nginx::

    Nginx ("engine x") 是一个高性能的 HTTP 和 反向代理 服务器，也是一个 IMAP/POP3/SMTP 代理服务器
    Nginx 是由 Igor Sysoev 为俄罗斯访问量第二的 Rambler.ru 站点开发
    Nginx 已经因为它的稳定性、丰富的功能集、示例配置文件和低系统资源的消耗而闻名
    BSD-like开源协议


PCRE安装包::

    为确保在 Nginx 中使用正则进行更灵活的配置，安装前需确定是否安装有 PCRE
    您可以到 ftp://ftp.csx.cam.ac.uk/pub/software/programming/pcre/ 下载最新的 PCRE 源码包

    
版本说明::

    Nginx 一般有两个版本，分别是稳定版和开发版. 目前为止1.0.x，1.2.x是稳定版；1.1.x，1.3.x是开发版
