正向代理&反向代理的区别::

    正向代理(forward proxy):
      可以发给任意 Web 服务器
      是 client 一端的服务
      对 client 来说，一般需要配置 proxy

    反向代理(reverse proxy):
      只能向事先设置好的目标进行转发
      是 server 一端的服务

    透明代理(transparent proxy):
      集合了正向代理和反向代理的优点
      更倾向于将透明代理说成是缓存
      对 client 是透明的
      ?: 现在和反向代理还分不分?
