如何演讲
########


* 演讲：表达的技术

一. 价值与效果
==============

写作的展现，是一种广度路线，产生间接、长尾效应；演讲的展现，是一种深度路线，产生直接、深度连接。


二. 经历与感悟
==============

成长路上，终究会遇上演讲；从没遇上演讲的程序员，可能天花板就会比较低。

而不同时长的演讲，准备的方式也不太一样。时间长的演讲，准备的内容就多，要精确地讲好这么多内容是一个挑战；而时间短的演讲，内容不多，但就需要合适地挑选和裁剪，并且精确地传递，这又是另外一种挑战。

三. 准备与发挥
==============

前期可以准备的内容::

    1. 框架
    2. 材料
    3. 节奏
    4. 表演


1. 框架
-------

::

    1. 目标：本次演讲需要达成的目标是什么？
    2. 听众：本次演讲的受众是哪些人？
    3. 重点：本次演讲要传递的关键点有哪些？


技术分享的框架线，可能有如下::

    1. 引出主题：结合目标与听众来确定。
    2. 自我介绍：让听众了解你，证明你有资格讲这个主题。
    3. 重点结构：每一个关键点的分析、讲解，可以从以下方面来拆解。
        a. 问题：这个点上存在什么问题？
        b. 历史：这个问题的历史由来是什么？
        c. 方法：你是用什么方法解决这个问题的？
        d. 原因：为什么要用这个方法，要在这个阶段，以及这样解决问题？
    4. 细节深入：有一定细节深入，更有说服力。
    5. 总结回顾：结束前的再次总结和提炼，以加深印象。

2. 材料
-------

材料包括三类::

    1. 幻灯片
    2. 演讲稿
    3. 小故事

1. 幻灯片的最大作用在于::

    a. 辅助演讲者的结构记忆与信息表达；
    b. 辅助听众的信息吸收、理解与消化。

    演讲的主角还是讲，而幻灯片仅仅是配角。

2. 演讲稿::

    讲之前你可以先写下来你所要讲的内容，这样会有助于组织信息、梳理逻辑和提炼语言。

    正常语速大约是每分钟 150～200 个汉字

3. 小故事::

    人是情感动物，故事的影响效应远高于数据和逻辑

3. 节奏
-------

关键的时间点::

    a. 开场
        据研究统计，一场演讲给人留下的印象和评价，开场的数秒至关重要。
        这可能和一开始是否能抓住听众的注意力有关。
    b. 峰终
        峰终定律（Peak-End Rule）
        在 “峰” 和 “终” 时的体验，主宰了对一段体验好或者不好的感受

4. 表演
-------

.. figure:: https://img.zhaoweiguo.com/knowledge/images/jikes/jinjies/lecture1.png

   表演准备的三个层级——from: Tim Urban’s Memorization Spectrum

做了前述准备的演讲，算是在框架内发挥。如果还准备了演讲稿，那么练习熟练后，基本算是接近了 3A 这个层级。好的演讲准备投入是巨大的，所以你一般需要判断到底多么重要的演讲，才需要用上 3 这个层级的准备。
