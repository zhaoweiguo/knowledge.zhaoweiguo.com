六步学习法
##########

.. figure:: https://img.zhaoweiguo.com/knowledge/images/theorys/learnings/6steps.jpeg

   其实几步不重要，重要的是构建一个不断上升的螺旋就可以了。


你想发展“技术 + 什么”::

    1. 技术 + 技术: 你技术很好，学新东西特别快，广度和深度都很好

       a) 技术 + 数据安全: 技术很好，对数据安全方面特别感兴趣

    2. 技术纵深不适合:

       a) 技术 + 行业: 解决方案专家
       b) 技术 + 售前: 技术型销售. 对公司产品有深入的理解，又懂客户、会聊天，

    3. 技术 + 管理: 有一定的领导力，喜欢和人打交道
    4. 技术 + 培训: 善于把自己所学传授他人
