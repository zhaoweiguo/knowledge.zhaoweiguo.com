tensorflowжҳҜз”ұJeff DeanйўҶеӨҙзҡ„и°·жӯҢеӨ§и„‘еӣўйҳҹеҹәдәҺи°·жӯҢеҶ…йғЁз¬¬дёҖд»Јж·ұеәҰеӯҰд№ зі»з»ҹDistBeliefж”№иҝӣиҖҢжқҘзҡ„йҖҡз”Ёи®Ўз®—жЎҶжһ¶гҖӮTensorFlow жҳҜи°·жӯҢзҡ„ејҖеҸ‘иҖ…еҲӣйҖ зҡ„дёҖж¬ҫејҖжәҗзҡ„ж·ұеәҰеӯҰд№ жЎҶжһ¶пјҢдәҺ 2015 е№ҙеҸ‘еёғгҖӮе®ҳж–№з ”з©¶еҸ‘еёғдәҺи®әж–Ү гҖҠTensorFlowпјҡејӮжһ„еҲҶеёғејҸзі»з»ҹдёҠзҡ„еӨ§и§„жЁЎжңәеҷЁеӯҰд№ гҖӢ

TensorFlow жҳҜ Google иҙҹиҙЈејҖеҸ‘зҡ„з”Ё Python API зј–еҶҷпјҢйҖҡиҝҮ C/C++ еј•ж“ҺеҠ йҖҹзҡ„ж·ұеәҰеӯҰд№ жЎҶжһ¶пјҢжҳҜзӣ®еүҚеҸ—е…іжіЁжңҖеӨҡзҡ„ж·ұеәҰеӯҰд№ жЎҶжһ¶гҖӮе®ғдҪҝз”Ёж•°жҚ®жөҒеӣҫйӣҶжҲҗ ж·ұеәҰеӯҰд№ дёӯжңҖеёёи§Ғзҡ„еҚ•е…ғпјҢе№¶ж”ҜжҢҒи®ёеӨҡзҡ„ CNN зҪ‘з»ңз»“жһ„д»ҘеҸҠдёҚеҗҢи®ҫзҪ®зҡ„ RNNгҖӮе…¶дјҳзјәзӮ№дёә:

вң“ е…·еӨҮдёҚеұҖйҷҗдәҺж·ұеәҰеӯҰд№ зҡ„еӨҡз§Қз”ЁйҖ”пјҢиҝҳжңүж”ҜжҢҒејәеҢ–еӯҰд№ е’Ңе…¶д»–з®—жі•зҡ„е·Ҙе…·;
вң“ и·Ёе№іеҸ°иҝҗиЎҢиғҪеҠӣејә;
вң“ ж”ҜжҢҒиҮӘеҠЁжұӮеҜј;

вң— иҝҗиЎҢжҳҺжҳҫжҜ”е…¶д»–жЎҶжһ¶ж…ў;
вң— дёҚжҸҗдҫӣе•Ҷдёҡж”ҜжҢҒгҖӮ

lenet-5жЁЎеһӢз®Җд»ӢпјҡLeNet-5жЁЎеһӢжҳҜYann LeCunж•ҷжҺҲдәҺ1988е№ҙеңЁи®әж–ҮGradient-based learning applied to document recognitionдёӯжҸҗеҮәзҡ„пјҢе№¶дё”жҳҜз¬¬дёҖдёӘжҲҗеҠҹеә”з”ЁдәҺж•°еӯ—иҜҶеҲ«й—®йўҳзҡ„еҚ·з§ҜзҘһз»ҸзҪ‘з»ңгҖӮLeNet-5жЁЎеһӢжҖ»е…ұжңүдёғеұӮгҖӮеҲҶеҲ«жҳҜеҚ·з§ҜеұӮ-жұ еҢ–еұӮ-еҚ·з§ҜеұӮ-жұ еҢ–еұӮ-е…ЁиҝһжҺҘеұӮ-е…ЁиҝһжҺҘеұӮ-е…ЁиҝһжҺҘеұӮгҖӮ

  • иҷҪз„¶DistBeliefе·Із»Ҹиў«и°·жӯҢеҶ…йғЁеҫҲеӨҡдә§е“ҒжүҖдҪҝз”ЁпјҢдҪҶжҳҜDistBeliefиҝҮдәҺдҫқиө–и°·жӯҢеҶ…йғЁзҡ„зі»з»ҹжһ¶жһ„пјҢеҫҲйҡҫеҜ№еӨ–ејҖжәҗгҖӮдёәдәҶе°Ҷиҝҷж ·дёҖдёӘеңЁи°·жӯҢеҶ…йғЁе·Із»ҸиҺ·еҫ—дәҶе·ЁеӨ§жҲҗеҠҹзҡ„зі»з»ҹејҖжәҗпјҢи°·жӯҢеӨ§и„‘еӣўйҳҹеҜ№DistBeliefиҝӣиЎҢдәҶж”№иҝӣпјҢе№¶дәҺ2015е№ҙ11жңҲжӯЈејҸе…¬еёғдәҶеҹәдәҺApache 2.0ејҖжәҗеҚҸи®®зҡ„и®Ўз®—жЎҶжһ¶TensorFlowгҖӮзӣёжҜ”DistBeliefпјҢTensorFlowзҡ„и®Ўз®—жЁЎеһӢжӣҙеҠ йҖҡз”ЁгҖҒи®Ўз®—йҖҹеәҰжӣҙеҝ«гҖҒж”ҜжҢҒзҡ„и®Ўз®—е№іеҸ°жӣҙеӨҡгҖҒж”ҜжҢҒзҡ„ж·ұеәҰеӯҰд№ з®—жі•жӣҙе№ҝиҖҢдё”зі»з»ҹзҡ„зЁіе®ҡжҖ§д№ҹжӣҙй«ҳгҖӮе…ідәҺTensorFlowе№іеҸ°жң¬иә«зҡ„жҠҖжңҜз»ҶиҠӮеҸҜд»ҘеҸӮиҖғи°·жӯҢзҡ„и®әж–Ү TensorFlow: Large-Scale Machine Learning on Heterogeneous Distributed Systems

docker:

$ docker pull tensorflow/tensorflow                     # latest stable release
