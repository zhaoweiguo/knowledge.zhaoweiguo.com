pyvis
=====

- 在 Python 中创建交互式网络图（图表）的库。
- 它基于 JavaScript 的 vis.js 库，能够生成漂亮且交互性强的网络图。
- pyvis 适用于需要展示和分析网络数据的场景，如社交网络、知识图谱、网络拓扑等。

* https://pyvis.readthedocs.io/en/latest/index.html
* https://github.com/WestHealth/pyvis
