Tavily Search
=============

* 官网: https://tavily.com/
* 文档: https://docs.tavily.com/docs/welcome
* Tavily Search API is a search engine optimized for LLMs, aimed at efficient, quick and persistent search results. Unlike other search APIs such as Serp or Google, Tavily focuses on optimizing search for AI developers and autonomous AI agents. We take care of all the burden in searching, scraping, filtering and extracting the most relevant information from online sources. All in a single API call!
* Tavily Search API 是一个针对优化的LLMs搜索引擎，旨在提供高效、快速和持久的搜索结果。与 Serp 或 Google 等其他搜索 API 不同，Tavily 专注于优化 AI 开发人员和自主 AI 代理的搜索。我们负责从在线资源中搜索、抓取、过滤和提取最相关信息的所有负担。全部在单个 API 调用中完成！

MindSearch
----------

* GitHub: https://github.com/InternLM/MindSearch
* Demo: https://mindsearch.openxlab.org.cn/
* MindSearch 是一个开源的 AI 搜索引擎框架，具有与 Perplexity.ai Pro 相同的性能。您可以轻松部署它来构建您自己的搜索引擎，可以使用闭源 LLM（如 GPT、Claude）或开源 LLM（如 InternLM2.5-7b-chat）。
* 上海AI Lab开源的AI search方案，能基于本地7B模型（InternLM2.5-7b-chat，上海人工智能实验室与商汤科技联合香港中文大学和复旦大学联合开发）
