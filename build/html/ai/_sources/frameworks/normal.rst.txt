第三代，即命令式深度学习框架，是由用类似NumPy的语法来定义模型的 `Chainer <https://github.com/chainer/chainer>`_ 所开创的。这样的思想后来被 PyTorch和MXNet的Gluon API 采用，后者也正是本书用来教学深度学习的工具。
