启动ssh隧道¶

linux下操作 [1]¶

  • 直接使用终端,输入:

    ssh -D 7070 username@yourserver.com

ssh -D 7070 -C -N myserver  # -C 开启压缩,-N 不执行命令

  • 使用gSTM客户端(ubuntu):

    apt-get install gstm
“应用程序->互联网“菜单下面找到:gSTM

备注

【优势】相对于其他代理工具来说，更加简单、易用，不需要安装和配置额外的软件，只需使用系统自带的 SSH 客户端即可。【劣势】相比Shadowsocks, 使用 SSH 协议所支持的加密方式和协议，无法使用更高级的加密和隐蔽技术，如混淆、流量伪装等，因此在一定程度上容易被 GFW 检测和干扰；只能通过 SSH 协议传输数据，因此在传输效率和速度上可能不如 Shadowsocks、Outline 等工具，特别是在高速、大流量的网络环境下。
