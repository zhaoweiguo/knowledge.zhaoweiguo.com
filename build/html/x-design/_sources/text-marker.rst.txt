屏幕画笔
########

关键词::

    on-screen annotation
    屏幕标注
    text marker

相关软件
========

* Hight: https://krugazor.eu/highlight/index.html.en::

    Modifiers:
        1. Free-form : no key down
        2. Line : ctrl key
        3. Oval : alt key
        4. Rectangle : shift key
        5. Grid line (horizontal or vertical) : ctrl + shift keys
        6. Arrow : ctrl + alt keys

    Shortcuts:
        1. Open/Save As, Quit : same as usual
        2. Delete last shape : delete (backspace)
        3. Clear everything : cmd + K
        4. Styles (preference panel) : cmd + alt + S

* Pointofix
* ScreenBrush: https://imagestudiopro.com/screenbrush/::
  
    持久化标识功能收费，只能显示1秒左右

* Mouseposé: https://alternativeto.net/software/mousepos-/about/::

    mouse pointer highlighting tool (aka "virtual laserpointer") 

* Pylote: http://pascal.peter.free.fr/pylote-en.html::

    dependencies to install Python 3 and PyQt5, see page install PyQt on Mac
