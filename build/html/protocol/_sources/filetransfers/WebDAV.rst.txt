WebDAV协议
##########

WebDAV （Web-based Distributed Authoring and Versioning） 一种基于 HTTP 1.1 协议的 通信协议。它扩展了 HTTP 1.1，在 GET、POST、HEAD 等几个 HTTP 标准方法以外添加了一些新的方法，使应用程序可对 Web Server 直接读写，并支持写文件锁定 (Locking) 及解锁 (Unlock)，还可以支持文件的版本控制。
